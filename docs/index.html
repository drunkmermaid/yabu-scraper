Yabu Scraper

⚠️ Módulo ECMAScript ! Não é possível utiliza-lo em projetos CommonJS, para mais informações acesse a documentação do Node.js.

Permite extrair informações sobre pesquisas, vídeos e da página inicial do website animeyabu.


Exemplo de implementação

import { search } from "yabu-scraper";

// Irá mostrar o título de cada anime retornado pela pesquisa no console.
async function printEveryTitle() {
    try {
        const results = await search("Naruto");
        for (const result of results) {
            console.log(result.title);
        }
    } catch (error) {
        console.log(error);
    }
}

printEveryTitle();
